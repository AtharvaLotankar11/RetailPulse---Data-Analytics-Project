## 🖥️ How to Capture Screenshots

### Method 1: Windows Snipping Tool (Recommended)
1. Press `Win + Shift + S`
2. Select "Window Snip" or "Rectangular Snip"
3. Capture the browser window
4. Save to `assets/screenshots/` folder

### Method 2: Browser Extension
1. Install "Full Page Screen Capture" extension
2. Click extension icon
3. Wait for capture to complete
4. Download and save to `assets/screenshots/`

### Method 3: Windows Game Bar
1. Press `Win + G`
2. Click camera icon
3. Screenshot saved to Videos/Captures folder
4. Move to `assets/screenshots/`

## ✅ Screenshot Requirements

- **Format:** PNG (preferred) or JPG
- **Resolution:** Minimum 1920x1080 (Full HD)
- **Quality:** High quality, no compression
- **Naming:** Use exact names listed above (lowercase, underscores)
- **Content:** Full page visible, no cut-off elements
- **Browser:** Chrome or Edge (clean UI)

## 🎨 Tips for Best Screenshots

1. **Maximize browser window** - Use full screen (F11)
2. **Hide bookmarks bar** - Ctrl+Shift+B
3. **Zoom to 100%** - Ctrl+0
4. **Wait for data to load** - Ensure all charts are rendered
5. **Clean sidebar** - Make sure logo and navigation are visible
6. **Good lighting** - If capturing screen with camera, ensure good lighting
