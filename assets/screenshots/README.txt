Screenshots Directory
=====================

This directory contains dashboard screenshots for documentation.

Required Screenshots:
1. home_dashboard.png - Home page with KPIs
2. sales_dashboard.png - Sales analytics page
3. customer_dashboard.png - Customer segmentation page
4. forecast_dashboard.png - Demand forecasting page
5. inventory_dashboard.png - Inventory management page

How to Capture Screenshots:
1. Run the dashboard: streamlit run dashboard/Home.py
2. Navigate to each page
3. Take full-page screenshots (use browser extensions if needed)
4. Save with the filenames listed above
5. Ensure screenshots are high quality (1920x1080 or higher)

Note: Screenshots should show the dashboard with actual data loaded.
